Generate RSA Key Pair (EI + LA + EO Commands)

Generates an RSA key pair inside the HSM. The private key never leaves the HSM boundary — it is returned LMK-encrypted and stored in HSM User Storage slot K000.

Sign Message (EW Command)

Signs a message using the RSA private key stored in HSM User Storage (K000). The HSM internally retrieves the key, decrypts it under LMK, computes the hash, and generates the signature.


Verify Signature (EO + EY Commands)

Verifies a signature using a public key. First imports the public key (EO) to get its MAC, then verifies (EY). Test with the correct key and a wrong key to see the HSM's different responses.
